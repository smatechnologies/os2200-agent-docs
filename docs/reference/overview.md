# Reference Overview

## What is it?

The reference section provides message catalogs for all OS 2200 LSAM components and a known-issues page. Use these pages when diagnosing unexpected job statuses, console messages, or communication errors.

## In this section

- [LSAM Job Messages](lsam-job-messages.md) — messages reported by the LSAM for OS 2200 batch job status
- [LSAM System Console Messages](lsam-system-console-messages.md) — messages the LSAM writes to the OS 2200 system console
- [LMAM Messages](lmam-messages.md) — LMAM status messages forwarded to OpCon and displayed at the system console
- [BIS MAM Messages](bis-mam-messages.md) — messages generated by the BIS MAM component
- [JORS Messages](jors-messages.md) — messages generated by the Job Output Retrieval System (SMAJOR)
- [XFRTCP Messages](xfrtcp-messages.md) — messages generated by the TCP/IP communications program
- [File Transfer Server Messages](file-transfer-server-messages.md) — messages from the SMAJOR FTServer component
- [File Transfer Agent Messages](file-transfer-agent-messages.md) — messages from the SMAFTA FTAgent component
- [Known Issues](known-issues.md) — documented limitations and workarounds

## How to use these pages

Each message page lists messages alphabetically or by category. For each message, the description explains the condition that produced it and, where applicable, the recommended action.

When troubleshooting an issue:

1. Identify which component produced the message (LSAM, LMAM, XFRTCP, JORS, or a file transfer component).
2. Navigate to the corresponding message page.
3. If the message is not listed, consult the [Known Issues](known-issues.md) page or contact SMA Support with the relevant log file.
